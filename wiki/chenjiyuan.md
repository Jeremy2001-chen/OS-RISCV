# RISCV64之旅

## RISC-V

### 一些Feature

RV64指令是64位的，交叉编译器可能会优化到32位。

### 寄存器列表

| 寄存器编号   | 编程接口名 | 补充说明                        |
|---------|-------|-----------------------------|
| x0      | zero  | -----                       |
| x1      | ra    | -----                       |
| x2      | sp    | -----                       |
| x3      | gp    | -----                       |
| x4      | tp    | Thread pointer              |
| x5-7    | t0-t2 | -----                       |
| x8      | s0/fp | -----                       |
| x9      | s1    | -----                       |
| x10-x11 | a0-1  | Parameter(2) & Return Value |
| x12-x17 | a2-7  | Parameter(8)                |
| x18-x27 | s2-11 | -----                       |
| x28-x31 | t3-6  | -----                       |
